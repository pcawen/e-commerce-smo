# Somnio e-commerce

Backend: NestJS (TypeScript) + Mongoose  
Frontend: AngularJS 1.x
Database: MongoDB

## Prereqs
- Node.js 18+
- MongoDB running locally (default URI in .env)

## Backend (api)
```
cd server
npm install
npm run start:dev
```


## Frontend (web)
```
cd client
npm install
npm start
```
